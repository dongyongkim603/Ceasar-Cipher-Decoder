CeasarCipher
by John Haney

An implemention of a simple program that takes in a ciphertext from the user
via the console and will perform either a decryption with a key or a brute 
force decryption. The results are then printed to both the console and to a
file named 'Results.txt'.

To run this project compile from the 'com' directory using the terminial abd the
command: 'javac com/MainDriverPackage/MainDriver.java' 
after compiling enter this command: 'java com.MainDriverPackage.MainDriver' to run 
the program.
